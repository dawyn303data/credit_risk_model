\# Credit Risk Default Prediction



\## Project Overview

This project builds a machine learning model to predict loan default risk using logistic regression.



The goal is to identify borrowers with high probability of default using financial indicators such as income, loan amount, and credit score.



\## Dataset

Loan default dataset containing borrower financial and credit characteristics.



Key features include:

\- Age

\- Income

\- Loan Amount

\- Credit Score

\- Debt-to-Income Ratio

\- Employment history



Target variable:

\- Default (0 = No Default, 1 = Default)



\## Tools \& Technologies

Python  

Pandas  

NumPy  

Scikit-learn  

Matplotlib  



\## Model

Logistic Regression with class balancing to address imbalanced dataset.



Evaluation Metrics:

\- Accuracy

\- Precision

\- Recall

\- F1-score

\- ROC-AUC



\## Results

The model identifies borrowers with higher default probability and visualizes model performance using ROC curve.
